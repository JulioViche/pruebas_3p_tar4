# 3p_tar4

Este repositorio contiene la solución a la actividad 4 del tercer parcial.

## Estructura del proyecto

- **src/**: Código fuente principal
  - `index.js`: Punto de entrada de la aplicación
  - `math.js`, `sum.js`: Módulos de utilidades matemáticas
- **test/**: Pruebas unitarias para los módulos de `src/`
  - `math.test.js`, `sum.test.js`
- **docs/**: Documentación del proyecto
- `package.json`: Dependencias y scripts del proyecto
- `eslint.config.js`: Configuración de ESLint

## Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/JulioViche/pruebas_3p_tar4
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```

## Uso

Ejecuta la aplicación principal:
```bash
node src/index.js
```

Ejecuta las pruebas:
```bash
npm test
```

## Linting

Para analizar el código con ESLint:
```bash
npx eslint src/
```

## Documentación

Consulta la carpeta `docs/` para instrucciones y el reporte de la actividad.
